# Creative Bank - Think Tank

## 1. 배경 및 목적

**프로젝트명**: Creative Bank

**제작 방식**: Claude Code(AI 코딩 도구)를 활용하여 직접 제작

**한 줄 요약**: 크리에이티브 기획자의 소재 기획 프로세스를 AI가 보조하는 업무 효율화 대시보드

### 만든 이유

소재 기획할 때 매번 같은 과정을 반복합니다.

> 앱 조사 → USP 도출 → 스토리보드 기획 → 기획서 작성해서 제작자에게 전달 → 데이터 분석

이 일련의 과정을 **기획자의 사고 흐름 그대로** AI가 보조할 수 있도록 하나의 툴로 만들었습니다.

- **업무 효율화**: 리서치/아이디어/기획서 작성에 드는 반복 작업을 AI가 대신
- **범용성**: 앱을 선택하면 그 앱에 맞는 대시보드가 나오는 구조 → 담당 앱 외 다른 앱 담당자도 활용 가능

## 2. 현재 구조 (4탭)

| 탭 | 핵심 역할 | 선행 조건 |
|---|---|---|
| 브레인스토밍 | 앱의 장르/USP/유저반응을 AI가 자동 분석 + 기획자 소구 방향성 메모 | 앱 선택 필요 |
| USP | USP별 카드로 정리 + 각 USP의 DA/영상 예시 스토리보드 미리보기 | 브레인스토밍 완료 필요 |
| 스토리보드 | USP + 포맷(DA/영상) 선택 → 스토리보드 자동 생성 + 베리에이션 | USP 선택 필요 |
| 데이터 분석 | CSV 업로드 → CTR/CPC/CPM 등 자동 계산 + 차트 | 독립 (선행조건 없음) |

### 핵심 플로우

| 단계 | 내용 |
|---|---|
| 01 | 앱 선택 (우파루 오딧세이, 한게임 앱 6개 등록) |
| 02 | AI 브레인스토밍 (장르/USP/유저반응 자동 분석) |
| 03 | 기획자 피드백 입력 ("이거 빼줘", "이거 추가해줘" 자연어 반영) |
| 04 | USP 카드 자동 생성 (USP별 DA/영상 예시 미리보기) |
| 05 | 스토리보드 생성 (USP + 포맷 선택 → 자동 생성) |
| 06 | 편집 + 베리에이션 (카피/이미지 수정, 변형 버전 비교) |
| 07 | 캡처하여 CD팀 전달 (스토리보드 그대로 디자인 요청 가능) |
| 08 | 데이터 분석 (CSV 업로드 → 성과 지표 자동 계산) |

## 4. 현재 한계 및 보강 필요 사항

| 구분 | 현재 상태 | 보강 방안 |
|---|---|---|
| AI 분석 | 사전 입력 데이터 + 패턴 매칭 (시뮬레이션) | LLM API 연동 → 실시간 분석 |

## 5. 변경사항 (2026-04-10)

### 방향 전환 (Pivot)

- **기존**: AI 기반 크리에이티브 기획 자동화 툴
- **변경**: **마케터-디자이너 간 소통을 위한 서비스 방향성 시각화 Tool**

### 전환 사유

- LLM API 연동 시 토큰 비용 부담 + 고도화 작업 필요
- 현 단계에서는 마케터-디자이너 간 협업 효율화에 집중하는 것이 실용적

### 기대효과 및 고도화 로드맵

| 순서 | 기대효과 | 상세 |
|---|---|---|
| 1차 | 서비스 방향성 공유 | 마케터가 정의한 서비스 방향성과 USP를 디자이너도 같은 툴에서 확인 → 이해도 향상 |
| 2차 | 기획-제작 워크플로우 통합 | 마케터가 별도 PPT로 스토리보드 만들지 않고 툴 안에서 작성 + 디자이너 피드백도 툴 내 진행 |
| 3차 | 성과 공유 | 소재 성과 데이터를 마케터뿐 아니라 디자이너도 공유받아 다음 제작에 반영 |

## 7. 우파루 오딧세이 — 서비스 방향성 & 크리에이티브 전략

### 7-1. 페르소나 정의

| 페르소나 | 연령/성별 | 차별화된 강점 | 사용 상황 | 니즈 | 문제점 | 플레이 스타일 |
|---|---|---|---|---|---|---|
| 수집형 전략가 | 18~34세 남녀 | 캐릭터 수집과 속성 조합으로 전략적 플레이 | 퇴근 후 여유 시간에 집중 플레이. 친구와 조합 공유 | 새로운 캐릭터를 모으는 재미와 속성 조합의 전략성 | 원하는 캐릭터 미획득/조합 깊이 부족 시 흥미 상실 | 탐구형, 연구형. 무작위성과 다양한 조합 실험 |
| 성장형 도전가 | 18~29세 남녀 | 캐릭터 육성 및 팀 구성의 성취감 | 여유 시간에 캐릭터 강화, 팀 구성 고민 | 캐릭터가 더 강해지고 랭킹에서 경쟁하고 싶음 | 성장 속도가 느리거나 목표 달성 어려우면 흥미 상실 | 목표 지향적, 꾸준한 투자로 점진적 성장 추구 |
| 힐링형 플레이어 | 25~34세 여성 | 방치형 시스템으로 스트레스 없는 성장 | 퇴근 후 스트레스 해소, 잠들기 전 힐링 | 자동으로 성장하며 편안함을 느끼고 싶음 | 너무 단순하거나 힐링 요소 부족 시 이탈 | 쉬운 접근성 선호, 최소 조작으로 결과 획득 |
| 꾸미기형 플레이어 | 20~30세 여성 | 건물 및 우파루 조합으로 나만의 땅 꾸미는 성취감 | 퇴근 후 자유시간 | 화려한 건물/장식/서식지를 수집하여 나만의 디자인 완성 | 과도한 현질 유도 시 흥미 상실 | 레어템, 새로운 디자인 추구 |

### 7-2. 페르소나별 메시지 & 소재 전략

| 페르소나 | 메시지 컨셉 | 소재 방향 | CTA |
|---|---|---|---|
| 수집형 전략가 | "속성 조합으로 나만의 우파루 팀을 완성하세요!" | 새로운 우파루 뽑기 등장 장면 / 물·불·바람 속성 조합 전투 연출 / 희귀 캐릭터 확률 강조 + 이벤트 | "지금 다운로드하고 속성 조합의 재미를 느껴보세요!" |
| 성장형 도전가 | "우파루를 키워 최강의 팀을 만들어보세요!" | 레벨업 전후 비교 / 강화된 팀의 전투 승리 장면 / 랭킹 상승 | "지금 시작하고 당신의 팀을 완성하세요!" |
| 힐링형 플레이어 | "자동으로 성장하는 당신만의 힐링 마을!" | 방치형 자원 축적 / 꾸미기로 완성된 마을 / 귀여운 캐릭터 상호작용 | "지금 다운로드하고 편안한 힐링을 경험하세요!" |
| 꾸미기형 플레이어 | "나만의 우파루 마을을 키워보세요!" | 레벨 1 평범한 마을 → 배속으로 레벨업 → 마을이 풍성해지며 고인물 느낌 나는 영상 | "지금 다운로드해서 당신의 마을을 키워보세요!" |

### 7-3. 페르소나별 키워드

| 페르소나 | 핵심 키워드 |
|---|---|
| 수집형 전략가 | 컬렉션, 레어템, 다양한 조합의 가능성 |
| 성장형 도전가 | 빠른 성장, 목표 지향, 레벨업 |
| 힐링형 플레이어 | 힐링, 무해하고 귀여운 우파루, 방치형 장점 |
| 꾸미기형 플레이어 | 무에서 유 창조, 다양한 디자인, 화려한 그래픽 |

### 7-4. USP / RTB 프레임워크

| USP (핵심 가치) | RTB (신뢰 근거) | 핵심 메시지 | 비주얼 코드 |
|---|---|---|---|
| **캐릭터** — 개성과 서사가 있는 600종 우파루 | 캐릭터마다 고유 설정/성격/외형, 속성별 다른 서식지, 시즌/이벤트 한정 캐릭터 | "논리는 없지만 귀여움은 확실한" | 캐릭터 단독 포커싱, 모션/표정, 서사 기반 상황 연출 |
| **컬렉션** — 조합해서 뭐가 나올지 모르는 재미 | 1,000+ 조합법, 11종 속성, 희귀/기간한정 우파루, 조합 결과 랜덤성 | "이 조합 해봤어?" | 조합 시도 → 결과 서프라이즈, 도감 빈칸 채워지는 순간 |
| **꾸미기** — 내 손으로 만드는 나만의 마을 | 5단계 마을 확장, 8종+ 서식지, 장식/건물 자유 배치, 농작물 재배 | "텅 빈 마을이 이렇게 변합니다" | Before→After 마을 변화, 배속 타임랩스, 풍성한 완성샷 |
